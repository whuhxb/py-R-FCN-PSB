1. 编译caffe，从此链接下载caffe，并编译:
git clone https://github.com/Microsoft/caffe.git
2. 参考编译好的caffe，直接放到文件夹py-R-FCN-PSB下面：
git clone https://github.com/whuhxb/Caffe-for-py-R-FCN-PSB.git
修改名字Caffe-for-py-R-FCN-PSB为caffe即可参照py-R-FCN训练py-R-FCN-PSB。
3. 数据集为NWPU-10 object detection dataset, 数据格式参照VOCdevkit0712，
下载链接参考相应link。
